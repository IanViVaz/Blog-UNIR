La Arquitectura del Mañana: Estructuras en HTML5

Explorando la ingeniería detrás de la web moderna y la importancia de la semántica digital.

En este laboratorio, hemos profundizado en el concepto de que el **HTML5** no es simplemente una herramienta para colocar texto en una pantalla. Es un lenguaje de arquitectura. La verdadera potencia de un sitio web profesional reside en su estructura invisible: la semántica.

1. El Esqueleto Semántico

Al utilizar etiquetas como <article>, <section> y <nav>, estamos comunicando a los motores de búsqueda y a las tecnologías de asistencia exactamente qué función cumple cada fragmento de contenido. Esto no solo mejora el SEO, sino que garantiza que la web sea universalmente accesible.

Concepto Clave: DOM (Document Object Model)

Cada vez que escribimos código HTML, estamos construyendo un árbol de objetos que el navegador interpreta. Durante esta actividad, aprendimos a manipular este árbol para crear una navegación fluida y lógica.

Para concluir esta fase, la integración de imágenes de alta resolución mediante hipervínculos externos nos ha permitido entender la eficiencia en el manejo de recursos distribuidos, manteniendo el repositorio de GitHub ligero pero visualmente impactante.
